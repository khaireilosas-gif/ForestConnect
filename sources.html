Data Sources & Acknowledgements

This project utilizes open data, APIs, and open-source software from the following organizations.

Primary Data Sources

  • Department of Statistics Malaysia (DOSM) Official national statistics for demographic and economic data.
  • Data.gov.my (MAMPU) The official Malaysian Public Sector Open Data Portal.
  • The Humanitarian Data Exchange (HDX) Open data for humanitarian and development indicators.

Geospatial Resources & Dashboards

  • OpenStreetMap (OSM) Open-source geographic data used for interactive basemaps.
  • USGS Earth Explorer / NASA Source for satellite imagery (Landsat) and digital elevation models (SRTM).
  • NICFI Satellite Data (Planet) High-resolution tropical forest imagery for environmental monitoring.
  • Esri ArcGIS Embedded spatial data dashboards used for visualizing deforestation metrics.

Artificial Intelligence & Academic APIs

  • OpenAlex API Open catalog of the global research system, used for retrieving scientific literature and localized forestry data.
  • Google Gemini AI Large Language Model utilized for processing academic data and powering the Forest AI retrieval-augmented chat engine.

Software & Technology Stack

  • Leaflet.js & Leaflet.heat The leading open-source JavaScript library for mobile-friendly interactive maps and heatmaps.
  • Supabase & PostgreSQL Open-source backend-as-a-service providing database infrastructure, image storage, and PostGIS spatial capabilities.
  • FastAPI (Python) High-performance web framework utilized for backend spatial intelligence and Getis-Ord Gi* automated clustering.

Media & Digital Assets

  • Pexels / Pixabay / Unsplash Royalty-free stock videos utilized for atmospheric backgrounds and educational impact demonstrations.
  • Google Fonts Typography integration (Montserrat).
